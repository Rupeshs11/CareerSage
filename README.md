## рҹҡҖ What is CareerSage?

CareerSage is a full-stack AI-powered platform that transforms how developers learn, practice, and compete. Instead of passively following tutorials, users get **personalized AI-generated roadmaps**, test their skills with **adaptive quizzes**, and challenge friends to **real-time 1v1 quiz battles** вҖ” all powered by NVIDIA's Llama 3.1 Nemotron model.

---

## вңЁ Key Features

| Feature                         | Description                                                                                                                              |
| ------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| рҹ—әпёҸ **AI Roadmap Generation**    | Enter a career goal and get a complete, step-by-step learning path with nodes, connections, and curated resources вҖ” all generated by AI. |
| рҹ§  **AI Skill Tests**           | Take AI-generated quizzes on any topic. Questions adapt to your level and include instant feedback with detailed explanations.           |
| вҡ”пёҸ **1v1 Rapid Fire Battles**   | Challenge friends to real-time quiz duels via WebSocket. Race to answer correctly, earn battle ratings, and climb the leaderboard.       |
| рҹ“Ҡ **Interactive Dashboard**    | Track your progress, quiz accuracy, battle stats, roadmap completion, and recent activity вҖ” all in one place.                            |
| рҹ‘Ҙ **Friends & Social**         | Add friends, view their profiles, send battle invites, and compete together.                                                             |
| рҹ”” **Real-time Notifications**  | Get instant notifications for battle invites, results, and friend requests via Socket.IO.                                                |
| рҹ”җ **Auth & Security**          | JWT-based authentication with bcrypt password hashing, change password support, and secure token management.                             |
| рҹ“ҡ **Resource Recommendations** | AI searches the web to recommend the best articles, tutorials, and courses for each step in your roadmap.                                |

## вҡЎ Quick Start

### Prerequisites

- **Docker** & **Docker Compose** installed
- **NVIDIA API Key** ([get one here](https://build.nvidia.com/))

### 1. Clone the repository

```bash
git clone https://github.com/Rupeshs11/CareerSage.git
cd CareerSage
```

### 2. Configure environment variables

```bash
cp backend/.env.example backend/.env
```

Edit `backend/.env` and fill in your values:

```dotenv
SECRET_KEY=your-secret-key
JWT_SECRET_KEY=your-jwt-secret-key
NVIDIA_API_KEY=your-nvidia-api-key
MONGODB_URI=mongodb://mongodb:27017/careersage
```

### 3. Run with Docker Compose

```bash
docker compose up -d
```

The app will be available at **http://localhost:5000**

### Local Development (without Docker)

```bash
# Install Python dependencies
cd backend
pip install -r requirements.txt

# Start MongoDB locally (must be running on port 27017)

# Run the dev server
python run.py
```

The app starts at **http://localhost:5000** with the frontend served directly by Flask.

## рҹ”„ CI/CD Pipeline

The project uses a **unified GitHub Actions workflow** (`pipeline.yml`) that handles everything:

| Action              | Trigger                 | What it does                                            |
| ------------------- | ----------------------- | ------------------------------------------------------- |
| `terraform-plan`    | Manual                  | Preview infrastructure changes                          |
| `terraform-apply`   | Manual                  | Create/update EC2 instance on AWS                       |
| `terraform-destroy` | Manual                  | Tear down EC2 instance                                  |
| `build-deploy`      | Manual / Push to `main` | Build Docker image вҶ’ Push to Docker Hub вҶ’ Deploy to EC2 |
| `full`              | Manual                  | Apply infra + build + deploy (end-to-end)               |

### Pipeline Flow

```
Push to main
    в”Ӯ
    в–ј
Build Docker Image в”Җв”Җв–ә Push to Docker Hub в”Җв”Җв–ә SSH Deploy to EC2
    в”Ӯ                       в”Ӯ                        в”Ӯ
    в””в”Җв”Җ Multi-stage         в””в”Җв”Җ Tagged with          в””в”Җв”Җ docker compose pull
        Dockerfile              SHA + latest              docker compose up -d
```

## рҹҗі Docker

### Build the image

```bash
docker build -t careersage:latest .
```

### Multi-stage build details

- **Stage 1 (builder):** Installs Python dependencies in an isolated layer
- **Stage 2 (production):** Copies only the compiled packages + app code
- Runs as **non-root** user (`careersage`)
- Built-in **health check** (`/api/health`)
- Production server: **Gunicorn** with eventlet worker for WebSocket support

## рҹҢҚ Infrastructure (Terraform)

Provisions an AWS EC2 instance with:

- **Security Group** вҖ” Ports: 22 (SSH), 80 (HTTP), 443 (HTTPS), 5000 (Flask)
- **gp3 EBS volume** вҖ” 20 GB, encrypted
- **User data script** вҖ” Auto-installs Docker + Docker Compose on first boot

```bash
cd terraform
terraform init
terraform plan -var="key_name=your-keypair"
terraform apply -var="key_name=your-keypair"
```
